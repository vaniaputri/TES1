Rangkuman PPM

Rangkuman
1. Git init : untuk menginisialisasi repository di .git
2. Git status : untuk mengetahui status
3. Git add namafile.txt: untuk menambahkan file baru ke Stagging Area
4. Git commit –m “teks” : menambahkan teks ke dalam file teks sebelumnya
5. Git commit –m ‘*…’ : berkomit di head,tapi belum repository jarak jauh
6. Git log : menampilkan daftar commit secara terurut
7. Git remote add origin : untuk mengirimkan perubahan ke server yang di tuju
8. Git push –u origin master : untuk mengirimkan perubahan ke repo local di GitHub
9. Git pull origin master : di GitHub dapat mengundang orang lain untuk melakukan perubahan dan dapat mengembalikannya ke awal sebelum terjadi perubahan yang di lakukan.
10. Git diff Head : untuk melihat perubahan terakhir di folder
11. Git add / : untuk menambahkan file.txt ke folder
12. Git diff –staged : untuk melihat perubahan di area stage, yang nantinya akan masuk ke commit
13. Git reset / : menghapus file
14. Git checkout --namafile.txt : mengembalikan perintah terakhir
15. Git branch clean_up : untuk menghapus branch (cabang)
16. Git checkout clean_up : untuk menukar branch(antar cabang)
17. Git rm ‘*.txt’ : untuk menghapus file tidak hanya di disk, tetapi juga menghapus file di stage
18. Git commit –m “…” : meng-commit perubahan yang telah di lakukan
19. Git checkout master : menganti ke master branch dan meng-copy perubahan di clean _up branch ke master branch
20. Git merge clean_up : untuk menggabungkan perubahan dari clean_uo branch ke master branch
21. Git branch –d clean_up : untuk menghapus branch clean_up
22. Git push : untuk mengirimkan semua perubahan ke remote repository
